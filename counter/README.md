# Счётчик просмотров книг приложения «Библиотека»

## Описание:
1. Приложение представляет собой микросервис, который опубликован как Docker-образ [здесь](https://hub.docker.com/repository/docker/professorseverussnape/counter/general).
1. Интегрируется с основным приложением «Библиотека» через docker-compose.
1. Обрабатывает 2 роута:
   - `POST /counter/:bookId/incr` - увеличивает значение счётчика на единицу,
   - `GET /counter/:bookId` - отдает текущее значение счётчика.
1. Данные счётчиков хранятся в Redis и сохраняются при перезапуске приложения или контейнера за счёт использования именованного Docker volume.
1. Если Redis недоступен (например, при локальной разработке без поднятого Redis), счётчики возвращают значение `0`.
1. Cкрипты в package.json предусматривают возможность упрощённого ввода docker-команд:
   - сборка development- и production- образов,
   - отправка production-образа на Docker Hub,
   - поднятие контейнеров из development- и production- образов,
   - поднятие контейнера с базой данных из образа redis последней версии с Docker Hub.
